## Hard Rules (baked into every agent)

- Every generated FastAPI backend exposes `GET /` and `GET /healthz`.
- Every generated Next.js frontend has a non-empty root page.
- `ANTHROPIC_API_KEY` is never stored in `.env` committed to git — use Doppler.
- GitHub repos are created PUBLIC (required for Vercel auto-deploy on free tier).
- Lemon Squeezy is the only payment processor. Stripe references are a build error.
- `gate=True` stages raise `PipelineBlockedError` on failure — they never return silently.
